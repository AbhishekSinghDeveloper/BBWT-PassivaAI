Basic Page Structure

All pages should have a title (displayed on the page) that matches the menu item used to select it.

All pages should have a simple description below the title.

Here is the basic layout of a page:

Page Title

...

HTML looks like this: 
            <div class="p-g">
                <div class="p-g-12">
                    <div class="card card-w-title">
                        <h1>Page Title</h1>
                        <div class="p-grid p-grid-responsive p-fluid">
                            <div class="p-g-12">
                                <label for="name" class="p-grid-col-2 control-label">Label</label>
                                <div class="label-content p-grid-col-3">
                                    <span class="p-float-label">
                                        <input pInputText id="textField" class="form-control"/>
                                        <label>Edit Text</label>
                                    </span>
                                </div>
                            </div>
                            <div class="p-g-12">
                                ...
                            </div>
                        </div>

                        <div class="p-grid-row">
                            <button pButton type="button" label="Save"></button>
                            <button pButton type="button" label="Cancel"></button>
                        </div>
                    </div>
                </div>
            </div>

Consider whether the page needs a description line. Description lines should be preferred when the use of a page is non-obvious, or when most of the user-base of the page will not be familiar with the page.
