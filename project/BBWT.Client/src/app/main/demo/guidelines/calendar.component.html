Date Time

HTML looks like this: 
            <div class="p-g">
                <div class="fixed-label">
                    <label for="calendar">Label</label>
                </div>

                <div class="p-g-12 p-md-4">
                    <p-calendar id="calendar" [(ngModel)]="calendarDate" [DateTimeValidator]="true"  showIcon="true" showTime="true" dateFormat="dd/mm/yy" (onInput)="dateTimeChange($event)"></p-calendar>
                </div>
            </div>
