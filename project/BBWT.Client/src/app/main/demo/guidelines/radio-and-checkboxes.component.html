Radio Buttons and Checkboxes

A set of radios each with their own label, to the right of the main label: 

                <div class="p-grid">
                    <div class="p-grid-row">
                        <label for="textField" class="p-grid-col-2 control-label">Label</label>
                        <div class="label-content p-grid-col-4">
                            <div class="radio">
                                <p-radioButton name="group1" label="Option 1"></p-radioButton>
                            </div>
                            <div class="radio">
                                <p-radioButton name="group1" label="Option 2"></p-radioButton>
                            </div>
                        </div>
                    </div>
                </div>

A set of radios with the options all on the same line:

                <div class="p-grid">
                    <div class="p-grid-row">
                        <label for="textField" class="p-grid-col-2 control-label">Label</label>
                        <div class="label-content p-grid-col-4">
                            <p-radioButton name="group1" label="Yes"></p-radioButton>
                            <p-radioButton name="group1" label="No"></p-radioButton>
                        </div>
                    </div>
                </div>

When in a vertical list of mixed controls, checkboxes should appear to the right of their labels. Where a checkbox is by itself, and especially if it requires a long sentence for its label then it consider sowing it on the left of its label.

HTML looks like this: 
                <div class="p-grid">
                    <div class="p-grid-row">
                        <label for="rightCheckbox" class="p-grid-col-2 control-label">Checkbox</label>
                        <div class="label-content p-grid-col-4">
                            <p-checkbox id="rightCheckbox" name="group1" binary="true"></p-checkbox>
                        </div>
                    </div>
                    <div class="p-grid-row">
                        <div class="p-grid-col-2"></div>
                        <div class="p-grid-col-4">
                            <p-checkbox id="rightCheckbox" name="group1" binary="true" label="This is a checkbox that ..."></p-checkbox>
                        </div>
                    </div>
                </div>
