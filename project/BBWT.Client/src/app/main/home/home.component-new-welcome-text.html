Welcome – BB Web Template Demo System


This is a demonstration site for BB Web Template, Blueberry’s platform for Web Development

BB Web Template is:
  • A high-level business-oriented web toolkit
  • A platform to accelerate web development projects
  • A long-term platform, building on the most popular libraries
  • Owned by BB but source-code licenced to customers
  • Security tested and robust
  • Deployable on Linux or Windows
  • Compatible with load-balanced deployment
  • Free of ongoing fees
 BB Web Template is not:
  • A way to lock in customers to Blueberry
  • Yet another trendy JavaScript Library
BB Web Template is built on:
  • Angular 8 – Open Source, Google
  • TypeScript – Open Source, Microsoft
  • .NET Core – C# - Open Source, Microsoft
  • PrimeNG – Open Source, Free GUI Toolkit
 BB Web Template delivers:
  • A long-term foundation for business
  • Scalability and Redundancy
  • Data Security
